# Llamaindex_test

Llamaindex 是一个用于构建和查询大型语言模型（LLM）的工具包。它提供了一个灵活的框架，可以与多种语言模型和数据源集成。

这是一个针对 Llamaindex 的测试项目。

## 环境

- Python 3.12.4
- Llamaindex 0.11.14

## 安装

pip3  install -r requirements.txt  

## 使用

```python3
python3 main.py
```
